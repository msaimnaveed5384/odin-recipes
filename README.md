# Odin Recipes

This project is part of **The Odin Project** Foundations curriculum.  
It is a simple recipe website built using basic **HTML**.

## Project Description

The goal of this project is to practice:
- Writing semantic HTML
- Creating multiple pages
- Linking pages together
- Organizing files and folders properly
- Using Git and GitHub with meaningful commits

The website contains a main index page with links to individual recipe pages.

## Project Structure
.
├── index.html
├── recipes/
│ ├── apple-pie.html

├── images/
│ ├── apple-pie-image.jpg
│ ├── Banana-Bread-Baked-Oatmeal.jpg

└── README.md


## Recipes Included

- **Apple Pie**
- **Banana Bread Baked Oatmeal**
- **Jeff’s Hotdog Chili**

Each recipe page includes:
- A title
- An image of the dish
- Recipe content
- A link back to the home page
